# FileTriage

FileTriage 是一个原生 macOS 文件快速分拣原型，适合整理桌面、下载、iCloud 文件夹和各种临时目录。

## 运行

```bash
cd apps/mac
swift run FileTriage
```

也可以直接运行：

```bash
./run.sh
```

## 打包给别人试用

```bash
./package_app.sh
```

打包结果会生成在：

```text
apps/mac/dist/文件快速分拣.zip
```

这是一个本地测试版，没有 Apple Developer 正式签名。别人第一次打开时，可能需要右键点击 App 选择“打开”，或在“系统设置 > 隐私与安全性”里允许打开。

## 当前功能

- 添加一个或多个来源文件夹，并扫描普通文件。
- 删除来源文件夹，并基于剩余来源重新扫描。
- 使用 macOS Quick Look 预览当前文件。
- 添加目标文件夹，一键把文件移动过去。
- 删除目标文件夹，并直接查看目标完整路径。
- 在预览正下方使用分拣按钮：目标文件夹、跳过、废纸篓。
- 使用右上角“撤销”按钮或 `Command-Z` 撤销上一步移动。
- 显示当前文件的创建时间、大小和完整路径。
- 支持在 Finder 中定位当前文件，或用默认 App 打开。
- 支持查看 iCloud 下载状态、下载云端文件、刷新预览。
- 使用 `Left` / `Right` 选择底部动作，按 `Return` 执行。
- 使用数字键 `1` 到 `9` 快速移动到前九个目标文件夹。
- 使用 Mac 的 `Delete` 键把当前文件移入废纸篓。
- 使用 `Space` 跳过当前文件。
- 可选扫描子文件夹和隐藏文件。

## 说明

当前版本可以用 Apple Command Line Tools 编译运行。后续如果要打包成正式 `.app`、添加图标、签名、权限和发布，Xcode 会更方便。
