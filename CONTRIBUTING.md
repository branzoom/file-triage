# Contributing

欢迎贡献想法、问题反馈和代码改进。

## 建议流程

1. Fork 这个仓库。
2. 新建一个分支。
3. 修改代码。
4. 运行对应平台的检查或构建命令。
5. 提交 Pull Request。

## macOS 检查

```bash
cd apps/mac
swift build
```

## Windows 检查

```bash
cd apps/windows
npm install
npm run check
```

## 反馈问题时建议提供

- 操作系统版本。
- 使用的是 macOS 版还是 Windows 版。
- 文件类型，例如图片、PDF、zip、rar、视频等。
- 复现步骤。
- 截图或录屏，如果方便的话。
