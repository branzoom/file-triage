# Contributing

**Languages:** English | [中文](#中文)

Contributions, bug reports, ideas, and design feedback are welcome.

## Suggested Workflow

1. Fork this repository.
2. Create a feature branch.
3. Make your changes.
4. Run the relevant checks.
5. Open a Pull Request.

## macOS Check

```bash
cd apps/mac
swift build
```

## Windows Check

```bash
cd apps/windows
npm install
npm run check
```

## Useful Details for Bug Reports

- Operating system version.
- Whether you are using the macOS or Windows app.
- File type, such as image, PDF, zip, rar, video, or text.
- Steps to reproduce.
- Screenshot or screen recording, if possible.
